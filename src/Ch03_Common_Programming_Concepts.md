# 常见编程概念


这一章会涵盖几乎每门编程语言中都会出现的一些概念，以及他们在 Rust 中的工作原理。许多编程语言在他们的核心处，都有许多共同点。出现在这一章中的概念都不是 Rust 所独有的，但我们会在 Rust 语境下讨论他们，并解释围绕使用他们的一些约定。

具体来说，咱们将了解

- 变量
- 基本类型
- 函数
- 注释
- 及控制流等。


在每个 Rust 程序中都将有这些基础知识，尽早掌握他们将给到咱们强大核心基础。

> **关键字，keywords**
>
> 与别的语言一样，Rust 语言也有一套保留仅供这门语言使用的 *关键字，keywords*。请记住，咱们不能将这些关键字用作变量或函数的名字。大多数关键字都有特殊含义，咱们将在咱们的 Rust 程序中使用他们完成各种任务；少数关键字没有与其相关的当前功能，但已针对今后可能添加到 Rust 中的功能而保留。咱们可以在 [附录 A](appendix/keywords.md) 中找到这些关键字的列表。
